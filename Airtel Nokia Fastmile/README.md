# VM Setup Instructions

Follow these steps to set up and run the virtual machine:

1. Download the `Tr369Server_VM.7z` file from this MEGA link:  
   [Download Tr369Server_VM from MEGA](https://mega.nz/file/ahJE0YgY#42sXhzKcle7z7m9H3tFgD1UHcp_sRkr6PF0GexlJMas)
2. Extract the downloaded `.7z` file into this folder.
3. Upon extraction, you will find:
   - A file named `Ubuntu_tr369_octopus.vmdk`
   - A folder named `Ubuntu_tr369_octopus`
4. Download and install **VMware** (such as VMware Workstation Player or Pro) if it is not already installed on your system.
5. Open the `Ubuntu_tr369_octopus` folder.
6. Simply double-click the `.vmx` configuration file inside the folder. This will automatically open and run the virtual machine in VMware.
